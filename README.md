# Exam-Pro

An online examination system built with PHP, designed for managing exams, candidates, and results.

## Features

- Admin panel for managing exams and users
- Candidate registration and exam attempts
- Examiner tools for creating and updating exams
- Secure login and password reset
- Complaint system
- Results viewing

## Installation

1. Clone the repository:

   ```
   git clone https://github.com/sivamsansiva/Exam-pro.git
   ```

2. Set up XAMPP or a similar PHP server.

3. Import the database:

   - Use the `db/examination_system.sql` file to set up the database.

4. Configure the database connection in `php/config.php`.

5. Place the project in your web server's root directory (e.g., `htdocs` for XAMPP).

6. Access the application via your browser.

## Technologies Used

- PHP
- MySQL
- HTML/CSS/JavaScript
- XAMPP (recommended for local development)
